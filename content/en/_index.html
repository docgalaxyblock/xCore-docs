{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} xCore combines the key components for your next robotics project onto a single board, making it easy to integrate low-level hardware. It features a high-performance STM32H723 microcontroller, a gigabit Ethernet switch, flash storage, an Inertial Measurement Unit (IMU), and a dedicated slot for the Raspberry Pi Compute Module 4 (CM4). All of this is packed into a compact board with a SODIMM connector, ready to fit into any custom project or robotics system. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-heart" title="Support Open Mower!" url="https://patreon.com/ClemensElflein" %}} Open Mower is a very complicated project, that's why the development and maintenance takes a lot of time.
